# Travel Solutions - UI Cards Project

Este proyecto es una prueba técnica para el puesto de Maquetador Frontend para la empresa Browser Travel Solutions.  
Está construido con **Next.js**, **React**, **SCSS** y utiliza **mixins y variables CSS** para mantener consistencia de diseño.

---

## Tecnologías

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- SCSS con **modular CSS** (`.module.scss`)
- Mixins y variables para diseño responsivo
- LocalStorage para tema claro/oscuro

---

## Requisitos

- Node.js v18+ (recomendado)
- npm v9+ o yarn
- Git (para clonar el repositorio)

---

## Instalación

1. Clonar el repositorio:

En bash ejecute lo siguiente:

git clone https://github.com/tu-usuario/nombre-del-repo.git

cd nombre-del-repo

2. Instale las dependencias

npm install

3. Ejecute en modo desarrollo

npm run dev

4. Diferentes accesos

Al estar corriendo la web esta generará dos link, uno en local y otro el cual podrá acceder desde otro dispositivo si desea probar el diseño responsive o también puede instalar la extensión de google llamada Responsive Viewer
